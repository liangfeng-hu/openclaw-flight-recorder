# Security Policy

## Scope
This repository is a research preview / PoC for local observability and log integrity.
It is NOT a malware scanner and does NOT provide security guarantees.

## Reporting a Vulnerability
If you believe you found a security issue in this repo:
1) Open a GitHub issue with minimal details OR
2) Contact the maintainer via the contact method you provide here: <YOUR_SECURITY_CONTACT>

Please do not include secrets (tokens/keys) in issues.

## Threat Model Notes
- This repo intentionally avoids “installer scripts” and does not ship editor extensions.
- Prefer local use and sharing only digests/summaries.
